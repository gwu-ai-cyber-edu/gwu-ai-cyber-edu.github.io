# Worksheet: ML Fit Spectrum

Use this worksheet during the Topic 06 discussion on matching cybersecurity tasks to appropriate AI/ML approaches.

## Instructions

Work in a small group. Generate cybersecurity tasks from your own teaching or professional context, choose 2-3 examples, and classify each task as a good fit for ML, possible with constraints, or poor fit for ML.

For each task, identify the likely type of ML, the main implementation or teaching challenges, and the role a human should play in the workflow.

## Reflection

- Which task was the strongest ML fit, and what evidence supports that choice?
- Which task looked tempting but had constraints that would make ML risky or ineffective?
- What would students need to understand before evaluating this task?
