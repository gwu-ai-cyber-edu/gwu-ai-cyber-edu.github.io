# Visual Adversarial Attacks

This is an open-ended exploratory lab designed to showcase tools and interactive demonstrations for exploring adversarial image generation.

### Assignment
1. Open and run each of the following adversarial image generation tools:
   - [ADVIS](https://jlin.xyz/advis/)
   - [adversarial.js](https://kennysong.github.io/adversarial.js/)
   - [XTransferBench-UAP-Linf](https://huggingface.co/spaces/hanxunh/XTransferBench-UAP-Linf)
   - [adversarial-tensorflowjs](https://krasch.io/adversarial-tensorflowjs/)
   - [TextAttack](https://github.com/QData/TextAttack)
   - [Adversarial Vision Playground](https://adversarialvision.owaisshariff.com/playground)
2. Pick one or more of these tools (or vibe-code your own) to generate a pair of original and adversarial images.
3. Post your generated image pair to the Slack channel along with your comments.
